API

Below is EmPANaDa's UI Application Programming Interface


ParameterDescription
live=[true/false] Turn continuous polling for the map. The map will poll every 5 seconds.
raw=[true/false] Turns on raw tweets. No filtering will be done. (experimental)
lat=[latitude]
lng=[longitude]
 Focuses the map on the desired location.
zoom=[1-20] Zoom to the desired level.
filter=[filters1,fiter2,...,filterN] Filters will be set accordingly.

Example:

http://empanada.cs.fiu.edu?raw=true&live=true
